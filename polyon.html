AnimatedPolylion

  • CLIENT: Polyon
  • PROJECT: Animated Polygon SVG
  • WEBSITE: http://polyon.co

Jonno Riekwel of Polyon enthusiastically approached Gray Ghost Visuals with a vision to develop and construct a vector animation sequence involving the Polyon logo wonderfully crafted by Jord Riekwel. Polyon requested that each polygon element animate indenpendently and in a staggered fashion in a desired sequence.

Noteworthy : The project was picked up for an in-depth guest article with CSS-Tricks due to an earier demo exploring the polygon lion orignally posted ion Dribbble by Breno Bettnecourt. This demo was exploring the capabilities of SVG polygons using the GreenSock Animation Platform.

  • Pre-Processors: Haml, Sass, CoffeeScript
  • Languages: HTML, CSS, JavaScript
  • Libraries: jQuery, TweenMax
